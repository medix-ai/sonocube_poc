# 이번 주 5일차 진행 보고 – 논문 리뷰 & 경량화 전략 수립

**보고 주제**: 2D→3D 초음파 관련 논문 리뷰, Pruning/Quantization 사례 조사, 사용 기술 방식 확정  
**기간**: 5일 (1일차~5일차)

---

## 1일차: 2D→3D 초음파 재구성 관련 최신 논문 리뷰 (1)

### 목표
- 2D echocardiography 기반 3D 좌심실 재구성/추정 분야 동향 파악

### 수행 내용
- **검색 키워드**: "2D echocardiography 3D reconstruction", "freehand ultrasound 3D", "LV volume estimation from 2D echo", "TSDF echocardiography"
- **리뷰 포인트**:
  - 2D 프레임만으로 3D 볼륨을 복원하는 방식(TSDF, NeRF 계열, 학습 기반 implicit 등) 정리
  - ECG/위상 정렬 의존 vs 비센서(이미지 기반) 방식 구분
  - 벤더/프로토콜 독립적 접근 사례 확인
- **산출**: 논문 목록 초안, 방식별 분류 표(센서 사용 여부, 입력 형태, 출력 형태)

### 보고용 한 줄 요약
2D 심초음파 기반 3D 재구성 관련 키워드로 논문 검색 및 방식을 “센서 의존 / 비센서” 기준으로 분류하여 정리함.

## 2일차: 2D→3D 초음파 재구성 관련 최신 논문 리뷰 (2)

### 목표
- TSDF/NeRF 및 딥러닝 기반 3D 복원 논문 심화 리뷰

### 수행 내용
- **TSDF**: 기존 영상처리/볼륨 퓨전 논문에서 truncation distance, voxel 해상도, 정렬 방식(ECG vs optical flow 등) 조사
- **NeRF/Implicit**: 단일 또는 소수 2D 뷰에서 3D 복원하는 최신 논문(의료/초음파 적용 사례) 요약
- **학습 기반**: 2D slice → 3D volume 예측, U-Net 3D, variational autoencoder 기반 복원 사례 파악
- **산출**: 방식별 장단점 비교표(정확도·계산비용·데이터 요구량·실시간성)

### 보고용 한 줄 요약
TSDF, NeRF/Implicit, 학습 기반 3D 복원 방식을 비교하여 정확도·계산비용·데이터 요구량 기준으로 정리함.

---

## 3일차: Pruning / Quantization 관련 사례 조사

### 목표
- 의료/초음파·엣지 배포 환경에서의 모델 경량화 사례 수집

### 수행 내용
- **Pruning**:
  - 구조적/비구조적 프루닝, L1/L2 기반 채널 프루닝, NAS와 연계한 사례 조사
  - 의료 영상 segmentation/classification에서 pruning 비율 대비 성능 유지 사례 확인
- **Quantization**:
  - PTQ(Post-Training Quantization) vs QAT(Quantization-Aware Training) 비교
  - INT8/FP16 적용 사례, ONNX Runtime / TensorRT 연동 시 정밀도·속도 개선 수치 정리
  - 초음파/심초음파 관련 양자화 적용 논문 또는 기술 보고서 검색
- **산출**: Pruning/Quantization 각 3~5건 요약표(방법, 데이터셋/도메인, 압축률, 성능 변화, 플랫폼)

### 보고용 한 줄 요약
Pruning과 Quantization 사례를 의료·엣지 관점에서 조사하고, 압축률·성능·플랫폼별로 표로 정리함.

---

## 4일차: 경량화 전략 수립 및 기술 방식 후보 확정

### 목표
- SonoCube PoC 적용 가능한 “2D→3D 재구성” 및 “경량화” 방식 후보 선정

### 수행 내용
- **2D→3D**:
  - 리뷰 결과를 바탕으로 1차 후보 선정 (예: TSDF 기반 퓨전 vs 학습 기반 3D 예측 중 택 1 또는 단계적 도입)
  - 데이터·라벨 가용성, 실시간성 요구사항, 비센서(ECG 독립) 요구 반영
- **경량화**:
  - 목표 플랫폼(macOS/Windows 데스크탑, CPU 우선 또는 GPU 가용 시)에 맞는 전략 수립
  - 1단계: ONNX 변환 + 런타임 INT8/FP16 양자화 시험
  - 2단계: 필요 시 QAT 또는 채널 프루닝 도입 검토
- **산출**: “사용 기술 방식” 초안 문서(2D→3D 방식 1안, 경량화 단계별 옵션)

### 보고용 한 줄 요약
논문 리뷰와 사례 조사를 바탕으로 2D→3D 재구성 방식 후보와 Pruning/Quantization 단계별 경량화 전략을 수립함.
